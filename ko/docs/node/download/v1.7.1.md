# 릴리즈 노트 <a id="release-notes"></a>

Klaytn v1.7.1은 트랜잭션 처리 중에 accountKey 필드를 변조시킬 수 있는 치명적인 메모리 손상 버그를 방지합니다.

**중요**: 모든 노드 운영자(CN/PN/EN 운영자)들은 노드 바이너리를 이 버전으로 업데이트해야 합니다.

## 수정
