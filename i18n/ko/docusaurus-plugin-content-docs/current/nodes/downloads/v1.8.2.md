### 수정사항
 - 이더리움과 동일한 receipt status를 반환하도록 수정된 `eth_getTransactionReceipt` API (#1257)
 - Raw 트랜잭션 API 반환에서 `EthereumTxTypeEnvelope`를 제거했습니다 (#1264).
 - `eth` 네임스페이스 API에서 존재하지 않는 블록에 대해 null을 반환하도록 수정했습니다 (#1261 #1267).
