## 클레이튼 v1.1.1 릴리스 노트

#### 수정 <a id="fixes"></a>

- Gini 계수 계산 버그를 수정했습니다.

#### 개선 사항 <a id="improvements"></a>

- 더 많은 테스트 케이스 추가
- 패키지 설명 업데이트
- 서비스 체인 브리지 컨트랙트에 저작권 메시지를 추가했습니다.
- 사용하지 않는 코드 제거

## 클레이튼 v1.1.0 릴리스 노트

### 새로운 기능 <a id="new-features"></a>

- 특정 블록 번호에서 스마트 컨트랙트 메서드 호출 지원
- 거버넌스 카운슬 멤버를 위한 다중 스테이킹 컨트랙트 지원

#### 수정 <a id="fixes"></a>

- 다양한 버그 수정

#### 개선 사항 <a id="improvements"></a>

- 라운드 수를 동기화하기 위해 거버넌스 카운슬 멤버에게 합의 메시지가 전파됩니다.
- 리팩터링된 패키지(보상, 거버넌스 등)
- doc.go에서 패키지 설명 업데이트
- 더 많은 단위 테스트 추가
