## 릴리스 노트 <a href="#release-notes" id="release-notes"></a>

이 버전에는 프로토콜 업그레이드를 위한 변경 사항이 포함되어 있으며, 2022년 1월 11일로 예상되는 블록 번호 #80295291 에서 Baobab 네트워크를 업그레이드할 수 있습니다. Baobab 테스트넷의 모든 노드는 명시된 블록 번호 이전에 이 버전으로 업그레이드해야 합니다.

### Baobab 네트워크 프로토콜 업그레이드

- 이더리움 London EVM 호환성을 위한 BaseFee EVM Opcode 도입(#1065, #1066, #1096)
- Baobab 네트워크에 대한 London EVM 하드포크 블록 번호 설정(#1080)

#### 새로운 기능

- PN 스팸 스로틀러 도입(#1092)
- 피어로부터의 원격 tx를 거부하는 txpool 플래그 추가 (#1075)
- 하드포크 블록 번호를 설정하도록 homi 업데이트 (#1070, #1082)

### 개선 사항

- solc v0.8 출력 형식 구문 분석 활성화(#1030)

### 수정 사항

- darwin/amd64용 Go 1.17을 지원하도록 golang syscall 라이브러리 버전 업데이트 (#1048)
- 노드 재시작 시 로컬 트랜잭션을 삭제하지 않도록 수정 (#1084)
- epoch 블록의 거버넌스 데이터 업데이트 지연 문제 수정 (#1089)

### 기타

- 보류 중인 블록 지원으로 원복(#1064)
