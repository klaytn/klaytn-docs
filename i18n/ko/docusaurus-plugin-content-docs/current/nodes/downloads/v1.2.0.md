## 클레이튼 v1.2.0 릴리스 노트

## 하이라이트 - 서비스 체인의 첫 번째 공식 릴리스 <a id="highlight-the-first-official-release-of-service-chain"></a>

이번 버전에서 서비스체인을 공식 지원하게 되었음을 발표하게 되어 기쁩니다.
서비스체인은 수평적 확장성을 달성하기 위해 설계된 Klaytn의 레이어 2 솔루션입니다. 서비스체인은 Klaytn 메인체인과 독립적으로 운영되며, 자체적인 거버넌스를 가질 수 있습니다. 서비스체인에 대한 자세한 내용은 [확장성 솔루션](../../learn/scaling-solutions.md) 에서 확인할 수 있습니다.

### 새로운 기능 <a id="new-features"></a>

- 서비스 체인 기능
- 데이터 앵커링: `ChainDataAnchoring` 트랜잭션을 통해 서비스 체인 데이터를 부모 체인에 앵커링할 수 있습니다.
- 크로스체인 밸류 전송: 서비스체인과 부모 체인 간에 KLAY와 KCT를 전송할 수 있습니다.
- 고가용성: 하드웨어/소프트웨어 장애로 인한 다운타임을 줄이기 위해 여러 개의 브리지 노드를 H/A 구성으로 설정할 수 있습니다.
- 서비스 체인을 위한 새로운 API
- klay_getDecodedAnchoringTransactionByHash
- subbridge_getParentOperatorBalance
- subbridge_getChildOperatorBalance
- subbridge_getAnchoringTxHashByBlockNumber
- subbridge_getValueTransferOperatorThreshold
- subbridge_setValueTransferOperatorThreshold
- subbridge_registerOperator
- subbridge_getRegisteredOperators

### 수정 <a id="fixes"></a>

- txpool의 경쟁 조건을 수정했습니다.

### 개선 사항 <a id="improvements"></a>

- 잠금 해제된 토큰만 체인 간 이동이 가능합니다.
