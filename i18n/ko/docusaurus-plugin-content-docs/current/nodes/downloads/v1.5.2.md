## 클레이튼 v1.5.2 릴리스 노트

### 새로운 기능
- 블록체인 데이터를 실시간으로 RDB에 푸시하는 체인 데이터 페처 구현 (#646)
  - 토큰 전송 이력을 로드하는 체인 데이터 페처 구현 (#646)
  - 내부/반전 트랜잭션 데이터를 로드하는 체인 데이터 페처 구현 (#646)
  - 컨트랙트 데이터 로드를 위한 체인 데이터 페처 구현 (#646)
  
- 서비스체인 노드에서 KAS 앵커 API 지원 구현 (#646)
  - 서비스 체인의 블록을 명령어 옵션을 사용하여 KAS 앵커 API를 통해 Baobab/Cypress에 쉽게 앵커링할 수 있습니다 (#646).
  - `txpool.allow-local-anchortx` 명령 옵션으로 JSON-RPC를 통한 앵커 트랜잭션 제출 허용 (#646)
  - 새로운 앵커 데이터 타입(JSON 형식) 지원 (#646)

### 기타
- 새로운 데이터베이스 구현(DynamoDB)이 추가되었습니다(#646).
- 제네시스 블록을 기존 상태로 초기화하기 위한 `overwrite-genesis` 플래그 추가 (#646).
- 다양한 유형의 캐시 구현을 지원하기 위해 stateDB 캐시 인터페이스 구현 (#646)
